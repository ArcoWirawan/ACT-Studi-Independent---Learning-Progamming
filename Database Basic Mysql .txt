Apa Itu Primary Key? ðŸ”‘

Primary Key adalah satu kolom (atau gabungan kolom) yang berfungsi sebagai identitas unik untuk setiap baris data.

Hukum alam Primary Key ada dua:

    UNIQUE (Unik): Nggak boleh ada dua data yang punya nilai yang sama di kolom ini. (Misal: NIK KTP. Nggak mungkin ada dua orang Indonesia yang NIK-nya sama persis).

    NOT NULL (Nggak Boleh Kosong): Wajib diisi. Nggak boleh ada data tanpa identitas.

Contoh di tabel Mahasiswa:

    nama: Bisa kembar (Budi Santoso ada banyak).

    jurusan: Bisa kembar (Anak TI ada banyak).

    nim (Nomor Induk Mahasiswa): Harus unik. Satu NIM cuma buat satu orang. Inilah kandidat terbaik buat jadi Primary Key.

tambahkan Query : UNIQUE KEY(nama_column);
Note : Harus ada column yang menggunakan PRIMARY KEY (supaya setiap data tidak sam aniknya)

    -> id INT NOT NULL AUTO_INCREMENT,

    -> nik INT(50) NOT NULL,
    -> PRIMARY KEY(id), # kolom id harus unik sebagai identitas urut warga #
